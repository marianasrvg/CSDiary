BACK TO BASIC


The basic day to day movement

git status

Check where you are and how you are. PD. Pls don't be at master

git add . | *

Add everything that you have change into a possible commit

git commit -m "message"

Commit your changes, if by accident you didn't write "-m" and weird text editor would open (aka VIM) just type :qw

git push

For when you are using Github and you want your changes to be in the remote branch


What you should do as soon as you start

git clone "url"
git checkout -b "newLocalBranch"

ADVICES

  • Never add or commit from master
  • Write good commit messages (they are going to save you)
